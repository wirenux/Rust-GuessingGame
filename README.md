# Guessing Game (Rust Training Project)

This repository contains a small command-line guessing game written in Rust.

It is a training project based on the official Rust book, *[The Rust Programming Language](https://doc.rust-lang.org/book/)*, and follows the learning approach used in the early chapters.

The goal is to practice core Rust concepts, including:
- Reading user input from standard input
- Parsing and validating input
- Pattern matching with `match`
- Using loops and control flow
- Comparing values with `std::cmp::Ordering`
- Generating random numbers with the `rand` crate
